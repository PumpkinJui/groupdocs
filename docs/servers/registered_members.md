# 服务器成员信息表

这里是测试群服务器的成员信息表。所有进入群服务器的成员信息由管理员登记在此。

信息表内记录以下信息：

- 群内称呼
- XboxID
- 生存服白名单状态
  - 生存服白名单须由管理员考核后给予
  - 开发服除管理员外不对外开放，信息标记在备注中
  - 测试服白名单不受生存服限制，在登记表中的成员未标注禁止进入的均可进入测试服
- 生存服成员墙代表物及颜色代码  
  成员如有意向，可以指定一个物品，放在成员墙上以代表自己。  
  代表物不得重复，必须在生存模式下可以获取，不得过于昂贵。
- 生存服成员墙状态
- 是否离群
- 是否违规

下表如有显示异常，请横屏查看。
