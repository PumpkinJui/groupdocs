# 矿山简介

:::tip

该文章的内容需要扩充。您可以帮助我们扩充更多信息。

:::

矿山是本服第二开始发展的区域，目前的定位为中世纪风格的小镇。

矿山一面临海，三面环山，因附近的裸岩山峰而得名。气候条件温暖，附近有稀树草原。

目前，矿山全境由 @文雨（KrisWenYu）和 @南瓜汁（PumpkinJui）管理，在矿山的部分活动需要和管理者报备。

## 发展史

矿山得名于其附近的裸岩山峰。因为裸岩山峰在高处会裸露出大量的矿物，因此这座山得名矿山。后来这个地区都被称呼为矿山。矿山是除新光外最先被发现的地区之一。

最初，@量筒 在新光发展的初期，为了做建筑而寻找云杉木。最终，他没有找到云杉木，但是先找到了矿山，并带走了很多的矿物，这为初期缺煤缺铁的新光解了燃眉之急，间接地促进了新光发展。然而，现在矿山的矿物已被消耗殆尽，随着新光工业的发展，来矿山挖矿也显得没有必要，于是矿山逐渐失去了它的本职工作。

后来，@南瓜汁 开始在矿山做进一步发展，并把矿山打造为了下界交通的头号大站，众多线路都经过矿山。应该说，正是 @南瓜汁 选择在这里发展，才使得矿山完成了从普通的资源型区域到发展区域的转型。@南瓜汁 在这里指导 @绿叶 建设了矿山基地，并建立了自己的仓库和轻工业体系。然而 @南瓜汁 的发展模式和一般的城镇不太一样，这里的机器等都是按高度而堆叠起来的，所以很多地方都显得尤为不便。@南瓜汁 戏称这是*纵向发展模式*。

矿山还曾见证群友第一次打凋灵，位置就在如今的莽田区。后来又在这里打了很多次凋灵，当时称为凋灵战场。但因为凋灵战场的地形破坏过于严重，后来就没有再在此处打过凋灵了。

随着 @南瓜汁 高考临近，矿山开始交由 @文雨 建设。新官上任三把火，文雨拆除了前任所留下的大多数建筑，并新建了一批新建筑，因为 @文雨 的建筑能力远远强于服务器内多数群友，以致老群友来到新矿山之后都纷纷直呼「这还是矿山吗？」。此外，@文雨 建设矿山期间也进行了下界交通大换血，矿山的线路被大幅简化，但仍然是举足轻重的换乘站，直到今天，矿山站通过了 3 条最长的下界交通线路，而这则是 @南瓜汁 打下的基础。矿山火车站（主世界）则是全服第二大的火车站。目前矿山地区在二人的合作下仍然在积极发展中。
