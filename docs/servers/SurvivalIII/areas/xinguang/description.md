# 新光简介

新光是本服务器出生点所在的城区，是全服第一大城镇，是本服最开始发展的区域。同时，新光也是服务器各大主要仓库的聚集地，和住宅区的聚集地，多数服务器玩家都长居于此。

新光主城区三面临海，除南村、筑心外的其他区域由多个岛屿组，并为 J 字型排布。这一定程度上导致了新光的桥多，发展也受到了地理条件的制约。

目前，新光全境由 @量筒（YZBWDLT）管理，在新光的部分活动需要和管理者报备。

## 城区规划

新光是一座由 6 区 2 县组成的城镇，不同的区县拥有不同的职能规划。

### 初晓区

**初晓区是新光的主城区**。新光由若干个岛屿构成，初晓区则为最中心的岛屿。世界出生点也位于初晓区，路面有金块做标记。

:::info[地图有待补充。]

:::

初晓区拥有大量的资源，一环内包括活动中心、新光站、屠龙纪念塔等公共建筑；二环内包括工作站、艺术馆（染坊）、市政厅、装备库、餐厅以及大量的民居；三环内则包括新光仓库、图书馆、公寓楼等。

> **小故事**：初晓区在六区规划的初期时被称作「新光区」，在一次群内大讨论时由 @巴豆 提出「初晓区」的名称，从此定名。

### 昌兴区

昌兴区位于新光的南部，是**新光的工业区，是大量生电工业设备的集中区域**。

:::info[地图有待补充。]

:::

> **小故事**：昌兴区原本的名字叫“交工区”，即“交易工业区”的缩写。后续在新光分区名称的讨论后，正式定下“昌兴区”的名字。

### 沐辉区

沐辉区位于新光初晓区北部，与初晓区隔河相望，是**新光的林场区域**。此处设有林场和陵园。

:::info[地图有待补充。]

:::

> **小故事**：关于沐辉区的起名由来：沐，有水有树。辉，指阳光的光辉，既点明了主题“沐辉”又有树木在阳光的光辉之下沐浴的情景。

### 浮霞区

浮霞区位于新光上空，**由一系列空岛设施组成**，与地面用水电梯相连。该区域最主要的设施是村民交易所。

:::info[地图有待补充。]

:::

> **小故事**：关于浮霞区的起名由来：出自其浮岛的本质及偏向新光西侧的位置。

### 随明区

随明区位于沐辉区以北，是**新光的畜牧产业集中区域**。全区位于北四环以北的位置。

:::info[地图有待补充。]

:::

> **小故事**：关于随明区的起名由来：主要含义是主区的附属位置。随有随和之义，明有明亮之说。

### 晨曦区

晨曦区位于初晓区以东，是**新光的农业区**。全区几乎位于东四环以东的位置。

:::info[地图有待补充。]

:::

> **小故事**：关于晨曦区的起名由来：晨曦区位于全城区的最东边，正对应日出于东。

### 筑心县

筑心县位于随明区以北较远的位置，计划建设一座用于便捷完成大量成就和高新区的新城，并用于逐步疏解主城区建筑分布过于密集所带来的卡顿问题。

:::info[地图有待补充。]

:::

### 南村县

南村县位于昌兴区以南的一座村庄附近。

:::info[地图有待补充。]
