# 新光未来规划

本页详细阐述了我们接下来的发展方向。如果你想参与到建设之中，请联系相关发起人。

## 长期规划

以下规划是正在且长期施行的规划方案。

### 图书大厦囤书计划

该计划由 @鸽子 发起，并由 @鸽子、@量筒、@巴豆 等人实施。计划在图书大厦大量囤附魔书以应对未来可能添加到游戏内的村民交易更改。

### 随明区养殖区完善计划

该计划由 @量筒 发起，并由 @量筒、@文雨、@十三酱 等人实施。计划在随明区继续完善养殖动物的种类数目。具体可能包含以下变动：

- [ ] 新建嗅探兽园；
- [ ] 新建狐狸与雪狐园；
- [ ] 新建北极熊园；
- [ ] 新建青蛙园；
- [ ] 新建兔子园；
- [ ] 新建骆驼园。

### 晨曦区种植区完善计划

该计划由 @量筒 发起，并由 @量筒、@十三酱 等人实施。计划在晨曦区继续完善种植农作物的种类数目。具体可能包含以下变动：

- [ ] 新建花田（包括火把花和瓶子草）
- [ ] 新建仙人掌田
- [ ] 新建蘑菇田
- [ ] 新建海带田

### 生肖广场同步计划

该计划由 @鸽子 发起。计划在生肖广场同步本年生肖的雕像。

## 近期规划

以下规划是将于近期施行的工作。

### 地铁建设计划：南村站

该计划由 @量筒 发起并实施。计划继续完善地铁线网，带动周边发展。当前进度：**南村**。具体可能包含以下变动：

- [ ] 贯通 4 号线南村 - 新光西站；
- [ ] 拆除 4 号线南村 - 新光西站老通道；
- [ ] 贯通南村路，打通从南村到地图画区域、印钞机区域的道路。

### 第三代猪人塔建设工作

该计划由 @小鼠 发起，并由 @小鼠、@绿叶 实施。计划迁移猪人塔的位置，并全面提升猪人塔的效率和功能。具体可能包含以下变动：

- [ ] 第三代猪人塔建设
- [ ] 第三代猪人塔调试
- [ ] 拆除第二代猪人塔

### 新光站重建计划

该计划由 @量筒 发起，并将由 @量筒、@文雨、@绿叶 实施。

新光站将在 2026 年春节后开始重建。目前已完成设计工作，设计者为 @文雨 和 @绿叶。新光站的重建将影响周边建筑设施，具体可能包含以下变动：

- [ ] 新原线改线；
- [ ] 新原线铁路桥从悬索桥改为由氧化铜块建成的桁架桥；
- [ ] 文雨家迁移；
- [ ] 硫化银家迁移；
- [ ] 重新规划人工矿道的入口；
- [ ] 市政厅迁移；
- [ ] 镐子石像迁移；
- [ ] 屠龙纪念塔迁移；
- [ ] 地铁 1 号线与 2 号线改线；
- [ ] 浮霞区东入口和中入口迁移；
- [ ] 篮球场迁移；
- [ ] 靶场迁移；
- [ ] 棋盘游戏迁移。

### 地铁建设计划：陵园站

该计划由 @量筒 发起并实施。计划继续完善地铁线网，带动周边发展。当前进度：**陵园**。具体可能包含以下变动：

- [ ] 拆除重建陵园站；
- [ ] 贯通 4 号线四环 - 陵园；
- [ ] 改善北四环辅路的道路逻辑，并将北四环辅路通车到晨曦、幻城

### 四环建设计划

该计划由 @量筒 发起并实施，计划在 2026 年内完成新光的四环（新光绕城高速）建设。具体可能包含以下变动：

- [ ] 完成东四环的建设和规划；
- [ ] 完成南四环的建设和规划；
- [ ] 完善北四环的道路；
- [ ] 增设昌兴出口；
- [ ] 增设晨曦出口。

### 浮霞区美化计划

该计划由 @量筒 发起并实施。计划使浮霞区的各空岛更美观且更安全。具体可能包含以下变动：

- [ ] 对村民交易所等添加空岛底座；
- [ ] 使用特制道路制式链接各空岛；
- [ ] 保障安全，防止玩家从空岛跌落。

### 黑石游戏机重建计划

该计划由 @量筒 发起，由 @量筒、@巴豆 实施。计划拆除初晓区南部的黑石游戏机并移动到别处。具体可能包含以下变动：

- [x] 拆除黑石游戏机；
- [ ] 建设到其他合适的位置。

### 三转炉重建计划

该计划由 @量筒 发起并实施。计划建设新的更大的熔炉组并拆除原有的三转炉。具体可能包含以下变动：

- [ ] 在三转炉的原位置附近和二代猪人塔附近建设新的更大编组的转炉
- [ ] 拆除原有的三转炉

## 远期规划

远期规划是已确定将要进行，但是近期内暂时不会实施的规划。

### 全怪塔建设计划

该计划由 @小鼠 发起。计划在史莱姆农场周边建设一个全怪塔，以弥补第三代猪人塔没有腐肉的不足。

### 筑心县建设规划

该计划由 @量筒 发起。计划在筑心县建立建立一片成就专区以便于玩家完成成就。同时，筑心县还将承接主城区的一些特殊建筑，以分担主城区过大的性能压力。

### 新光西站重建计划

该计划由 @量筒、@比翼鸟 发起。计划重建新光西站以适应未来的需求。具体可能包含以下变动：

- 重新设计新光西站的总体构造
- 重新设计新光西站的地下层，以便于搭乘地铁
- 将新光西站扩展为至少 3 台 6 线的大站点
- 将昌兴轻轨接入

### 药店建设计划

该计划由 @量筒 发起。计划将工作站改造成药店。具体可能包含以下变动：

- 将工作站改造成存储药水及其原材料的仓库；
- 将工作站改造成含有大量酿造台的专项工作站。

### 公寓楼完善计划

该计划由 @量筒 发起。因为当前的工作站通过脚手架上下楼，有所不便，之后可能要对公寓楼做一些改进。具体可能包含以下变动：

- 添加楼梯和水电梯。

### 新光南站建设计划

该计划由 @量筒 发起。计划在当前印钞厂附近设立一个新的火车站：新光南站，并由新原线通过。计划设定为 1 台 2 线的小站点。

### 快速路高速路信标安装计划

该计划由 @量筒 发起。计划在快速路和高速路附近安装满级速度信标。

## 超远期规划

超远期规划是暂时不确定是否执行的规划方案，或者仅为对未来的一种大体构想的规划。这些规划方案仍处于非常早期的状态，仍需验证可行性之后再进一步动工。

### 新光东站建设计划

该计划由 @量筒 发起。计划在当前晨曦区东侧设立一个新的火车站：新光东站。目前暂时不确定通过线路、站台规模，仍需后续发展后做进一步判断。

### 活动中心改造计划

该计划由 @比翼鸟 发起。计划将活动中心改造为博物馆性质的建筑。

### 林场分类中心改造升级计划

该计划由 @量筒 发起。旨在同步后续版本推出的新树木，和下界菌种。
