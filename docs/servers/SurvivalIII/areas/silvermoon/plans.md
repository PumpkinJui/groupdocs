# 银月未来规划

## 长期规划

以下规划是正在且长期施行的规划方案。

## 近期规划

以下规划是将于近期施行的工作。

### 银月二厂设施搬迁

拟将目前位于皓月区中心地带的银月第二工厂内设施外迁至银月新港（坐标：-1412, ?, -1160）区域，建设要求：

1. 外墙材料自备。
2. 建筑风格任选，可选用石料、木板、混凝土等材料。但建造成果需要保证起码的美观性。
3. 建筑大小不得过大。不得超出道路范围。
4. 只需将二厂内的设施（两台转炉、两个无限熔岩源、一座大型自动化农场（可拆可迁）全部搬走，原建筑无需拆除。

## 远期规划

远期规划是已确定将要进行，但是近期内暂时不会实施的规划。

## 超远期规划

超远期规划是暂时不确定是否执行的规划方案，或者仅为对未来的一种大体构想的规划。这些规划方案仍处于非常早期的状态，仍需验证可行性之后再进一步动工。

### 银月全境照明工作

该计划由 @量筒 发起。为解决银月全境及周边和银月地铁周边的刷怪问题，需要做好充足的照明工作，确保玩家在附近空闲时不会被怪物袭击。

### 银月地铁理线工作

该计划由 @量筒 发起。为解决银月地铁站距过短、线路布局过于混乱的问题提出此规划。

### 银月地铁与月矿线共线段正线化工作

该计划由 @量筒 发起。为解决银月地铁与月矿线共线段强制减速的问题提出此规划。
