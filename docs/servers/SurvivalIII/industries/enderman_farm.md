# 末影人农场（小黑塔）

末影人农场是基于末影人对末影珍珠砸出的末影螨的仇恨原理而制成的，量产末影珍珠的机器。

![小黑塔](/servers/SurvivalIII/industries/enderman_farm.png)

## 基础信息

- **地点** 末地主岛中心附近（20, -25）
- **维修者** 十三酱
- **主要产物** 末影珍珠
- **到达方式** 末地主岛
- **代际** 第一代

## 使用方法

按引导跳进处死区，近战攻击以处死。
