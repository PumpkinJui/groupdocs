# 主世界轨道交通与城际铁路

主世界拥有庞杂的轨道交通系统，以连接世界各地。目前，主世界轨道交通一共有 3 套轨道交通系统，总里程达到 16338 米。

- [新光地铁](./areas/xinguang/railway)：主要服务于新光及周边地区，一共拥有 5 条线路，设站 19 座，总里程为 5421 米。
- [银月地铁](./areas/silvermoon/railway)：主要服务于银月及周边地区，一共拥有 7 条线路，设站 33 座，总里程（非共线）为 4385 米。
- 城际铁路：主要连接世界各地，一共拥有 3 条线路，设站 11 座，总里程为 6532 米。

### 站点间距

> 以下测距点的选取按照如下标准执行：
>
> 1. 取其中一条轨道，另一条轨道不计入里程；
> 2. 对于非终点站站点，取减速动力铁轨的中点；
> 3. 对于终点站站点，取站点端点，不取折返线长度；
> 4. 得到两个测距点(x1, z1)和(x2, z2)，按照|x2-x1|+|z2-z1|得到长度值。
