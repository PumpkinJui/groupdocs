# 守卫者农场

守卫者农场是基于海洋神殿刷新守卫者的机制而制成的机器，以量产海晶砂粒和海晶碎片。

![守卫者农场](/servers/s3/industries/guardian_farm.webp)

## 基础信息

- **地点** 新光东南方向（215, 215）
- **维修者** 小鼠
- **主要产物** 海晶砂粒、海晶碎片、生鳕鱼
- **到达方式**
  - **下界交通**：绿线，前往守卫者农场站并根据导引到达
  - **城际铁路**：月矿线，前往守卫者农场站上楼到达
  - **鞘翅**
- **代际** 第一代

## 使用方法

按照机器内路牌引导，前往机器上方挂机即可。
