# 冒险世界：苏醒

冒险世界：苏醒是一张大规模的地图。该地图为综合向解谜地图，涵盖跑酷、PVE、解谜等各种玩法。玩家在其中扮演「林乐」的角色，展开一场封印魔王的故事......
