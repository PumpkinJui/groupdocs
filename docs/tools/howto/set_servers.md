# 如何设定自定义服务器

:::info

本文不讨论 Xbox、Nintendo Switch 和 PlayStation 上的设定方式。

某些信源表明，这些平台暂时无法直接设定自定义服务器，但确实可以通过技巧操作完成设定，包括修改 DNS 和使用 BxTx 一类第三方应用等方式。请读者自行判断这些方法的稳定性与安全性。

:::
:::info

原文内容来自 [Play Minecraft: Bedrock Edition Online in a Multiplayer Server](https://help.minecraft.net/hc/en-us/articles/4410316619533-Play-Minecraft-Bedrock-Edition-Online-in-a-Multiplayer-Server)，并经 AI 翻译和人工调整。

:::

1. 启动《Minecraft》，确保已经登录 XBox 账号。
2. 在标题屏幕中选择「游戏」或「开始游戏」。
3. 选择「服务器」标签页。
4. 选择「添加服务器」。
5. 输入服务器名称。此项可以随便填，只是用来让你知道这是什么服务器。
6. 输入服务器地址。服务器地址可以是 IP（如 `127.0.0.1`）或 URL（如 `www.example.com`）。我们通常给出一个 URL，但完整信息可以在设定页面查询。
6. 输入服务器端口。默认端口为 19132，但群内服务器通常不使用此端口，请以实际设定信息为准。
7. 选择「添加并开始」，或选择「添加服务器」以便稍后游玩。
