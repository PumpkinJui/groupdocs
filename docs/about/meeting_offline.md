# 面基进度

## 面基三定律

在我们的实际面基经验中，我们创立了「*面基学*」并总结出了三条定律：

> **面基学第一定律**：面基只有零次和无数次，除非某次面基不愉快。  
  **面基学第二定律**：当和某人或者某群人面基十分愉快时，会产生和 TA 或者 TA 们再次面基的意愿。  
  **面基学第三定律**：面基行为具有感染性。一个人的面基行为会鼓动身边的人进行面基。

*（量筒：怎么会有人义正言辞地把这种东西写出来啊可恶！）*
