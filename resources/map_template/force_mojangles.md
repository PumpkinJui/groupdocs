# 强制 Mojangles 字体 文档

## 包性质

本包为资源包。

## 使用说明

* 装载本包之后，将在您的世界强制使用 Minecraft 风格的英文、数字和 § 字体。
  * 这类似于在 Java 版中关闭「强制使用 Unicode 字体」的效果。

* 装载此包时需要注意以下事项：
  * 在退出装载了此包的地图后，您可能会看到英文、数字等被更改的字体在转换后会以不正确的间距显示的问题。这问题可通过重启游戏解决，如果您不能忍受此问题，请勿装载此包。
